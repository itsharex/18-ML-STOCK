## 🚨 一、一票否决（铁律）

触碰以下任一条，直接放弃，不再查看其他指标。

### 1.1 审计师相关

| 规则 | 说明 |
|------|------|
| **年报前更换审计机构** | 尤其是年报审计启动前（Q4），将合作多年的老所换新所。原审计师不愿再为报表背书，是最危险的信号 |
| **审计报告非标意见** | 带强调事项段、保留意见、无法表示意见 → 财务已不可信，无论股价如何都别碰 |
| **核心财务负责人频繁更换** | 一年内财务总监/审计部负责人换两次以上 → 内部账务存在无法调和的矛盾 |

### 1.2 合规与内控

| 规则 | 说明 |
|------|------|
| **资金占用/违规担保诉讼** | 出现立案调查或诉讼纠纷 → 立即退出，不抱任何幻想 |
| **印章失控传闻** | 即便只是传闻也应视为最高警报 → 内控已完全崩塌 |

## 📋 二、盈利质量核心检查

这部分检验"稳定盈利"究竟是真实现金流，还是纸面上的会计数字。

### 2.1 核心指标对比（逐季度/逐年追踪）

#### 经营现金流净额 ÷ 净利润

| 阈值 | 判断 |
|------|------|
| > 0.8 | ✅ 健康 |
| 0.6 ~ 0.8 | ⚠️ 注意 |
| **< 0.6** | 🔴 **高度警惕** |
| **< 0.3** | 💀 **财务操纵重灾区** |

**计算方式**：年报现金流量表中"经营活动产生的现金流量净额" ÷ 利润表中"净利润"

**含义**：连续两年低于 0.6，或单年出现正值但经营现金流为净流出 → 利润只是纸面数字，没变成现金收回来。

#### (应收账款 + 存货) ÷ 总资产

| 阈值 | 判断 |
|------|------|
| < 20% | ✅ 健康 |
| 20% ~ 30% | ⚠️ 注意 |
| **> 30%** | 🔴 **高度警惕** |

**含义**：该比例超过 30% 并逐年攀升 → 公司靠垫款和压货维持收入增长，坏账和减值风险极高。

**特别关注**：应收账款增幅是否常年大幅超过营业收入增幅 → "提前或虚假确认收入"的典型征兆。

#### 折旧费用异常

对比当年折旧费用 ÷ 固定资产原值：

- 远低于行业惯例 → 公司可能通过拉长折旧年限来虚增利润

#### 大额减值对比

- 同行业其他公司均计提减值，而标的常年不计提 → 会计政策过于激进，利润虚胖

## 👀 三、大股东与内部人行为

如果基本面数据让你犹豫不决，看内部人的行动往往能一锤定音。

### 3.1 股权质押

| 阈值 | 判断 |
|------|------|
| **> 70%** | 🔴 **高度警惕** |
| 50% ~ 70% | ⚠️ 注意 |

**含义**：质押超过 70% 且股价下跌 → 大股东偿债压力大，挪用上市公司资金的动机大幅增加。

### 3.2 减持与增持

| 行为 | 信号 |
|------|------|
| **大股东/高管大幅减持** | 🔴 极度负面 — 尤其在估值高位或业绩欠佳时，他们在用自己的钱投票 |
| **内部人自筹资金二级市场买入** | ✅ 正面信号 |
| **极小金额象征性增持** | 信号意义大打折扣 |
| **员工持股计划/股权激励形式增持** | 信号意义有限 |

## ⏰ 四、持续性监控

| 动作 | 频率/时机 |
|------|----------|
| **年报前 10 天重新扫一遍所有条目** | 每年固定时间点 |
| **每季度报表出来后过一遍** | 每季度 |
| **异常换手观察** | 停牌或复牌前后 — 基本面未改善的情况下异常放量，别跟风抄底 |
| **信披查原始公告** | 去深交所官网 / 巨潮资讯网(cninfo.com.cn)查原始公告，不依赖二手信息 |

## 📐 使用流程

```
每次买入前：

Step 1: 一票否决 → 触碰任意一条 → ❌ 停止，不买
                → 都没触碰 → 进入 Step 2

Step 2: 盈利质量体检 → 核心指标全部通过 → 进入 Step 3
                     → 有指标亮红灯 → ❌ 停止

Step 3: 内部人行为检查 → 无异常 → ✅ 符合条件，可考虑
                       → 大比例减持/超高质押 → ❌ 停止
```

## ⭐ 作者核心教训

1. 看似"每个季度都稳定盈利"的公司，现金流可能只是纸面数字
2. "概念会轮动回来"不能当持仓的护身符 — 基本面暴雷时任何概念都会瞬间失效
3. **换审计师是最致命的先行信号**，比任何财务指标都值得重视
4. 投资逻辑需要分层：
   - **守**（财务质量、审计意见、管理层诚信）→ 决定能不能拿
   - **攻**（行业概念、热点）→ 决定盈利空间
   - 守的部分受损，无论什么概念都应立即止损
